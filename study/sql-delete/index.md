> DELETE

<br>

- 테이블의 행을 삭제할 때 사용하는 구문

<br>

``` sql
-- DELETE 구문 형식
DELETE FROM TABLENAME
WHERE 조건;

-- FROM 키워드는 생략가능하다.
-- WHERE 조건; 없이 사용하면 테이블 전체 삭제되기 때문에 주의해야한다.
```
