> 멀티 스레드, Multi-Thread

<br>

- 프로그램, Program
  - 실행 파일 (exe 파일)
  - 하드디스크에 저장되어 있는 상태
- 프로세스, Process
  - 실행 중인 프로그램
  - 메모리(RAM)에 실행 상태로 로드된 프로그램
- 스레드, Thread
  - 프로세스의 작업 단위(실행가능 단위)
- 멀티스레드, Multi-Thread
  - 스레드가 여러개인 프로세스 상태

<br>

약간 다른 개념들이지만

- 멀티 프로세스, Multi Process
  - 프로세스가 여러 개인 프로그램
- 프로세서, Processor
  - 프로세스를 동작시키는 장치, CPU
- 멀티 프로세서, Multi-Processor
  - 다중처리 컴퓨터
  - 프로세서가 여러개인 시스템(컴퓨터)
- 멀티 코어 프로세서
  - 코어가 여러개인 프로세서
  - 듀얼 코어, 쿼드 코어, 헥사 코어 .. 등등
