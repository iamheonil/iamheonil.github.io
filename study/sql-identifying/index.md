> 식별관계, 비식별관계

<br>

PK, FK 의 관계

- 식별 관계(Identifying Relationship)
  - FK가 해당 테이블에서 PK 역할을 수행하는 것
- 비 식별 관계(Non-Identifying Relationship)
  - FK가 해당 테이블에서 PK 역할을 수행하지 않는 것
