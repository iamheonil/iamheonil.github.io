> JavaScript의 특수한 자료형

<br>

- Undefined
  - 변수가 초기화되지 않은 상태
  - 변수의 값이 할당되지 않은 상태
  - 변수가 선언되지 않아서 아예 오류가 나는 상황과는 다른 상황인걸 인지해야 한다.
  - 데이터타입조차 결정되지 않은 상황
- Null
  - Object 타입의 데이터
  - 키워드
  - 참조대상이 없음을 나타냄
  - 변수를 생성하고 의도적으로 값을 비워둔 상태
  - Undefined 는 의도적이지 않고 Null 은 의도적이라고 볼 수 있다.
